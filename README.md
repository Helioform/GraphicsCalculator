# GraphicsCalculator
3D graphics calculator

# How to use

To display any 3d function.
    
    Examples: Math.cos(y)+Math.cos(x)
              Math.sin(Math.sqrt(x*y))
              Math.sin(x)*Math.cos(y)
              Math.sin(Math.sqrt(x*x+y*y))
              x
              y
              x*y
              x*x+y*y
